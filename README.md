# Great Thinks Thinkspace
This repo is a workspace for the Great Thinks podcast. Its purpose is to act as a centralized spot to collaborate on podcast ideas and discuss potential topics, format, production, schedule, or just anything at all related to the podcast.

**Repo Files**: Used to archive episode notes.

**Wiki Pages**: Brainstorm general ideas and store podcast format documentation.

**Issues**:
* Outline and build individual episode notes. Use checklist to track topics live while recording.
* Raise issues that require discuss to make a decision.

**Projects**: If we really want to get granular, we can use the a Projects board to develop and schedule Episode contents.
